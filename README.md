# Springify
A Spring based music streaming application

## Goals
- [ ] Create a Login with SpotifyAPI
- [ ] CRUD Enabled Playlists
- [ ] Stream Music from Spotify Catalog via SpotifyAPI
- [ ] Play/Pause/Skip Songs
- [ ] Display Song Metadata to User
